Projects

I built a booking website for my acapella group!

As the Resource Manager for NUS Resonance, I was tasked to manage the practice booking timeslots for members. The web app allowed members to submit a timeslot requests to me. Once I approve the request, an automatic email is sent to the member for confirmation using EmailJS. I built it using React, Google Calendar API and Firebase.
